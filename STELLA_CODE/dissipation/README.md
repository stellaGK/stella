# Dissipation

This directory contains the collisions and dissipation terms. 

- dissipation_and_collisions.f90 -> Adds the collisions or hyper dissipation options for stella

Collision options: 

- collisions_dougherty.f90 -> dougherty model for collsisions
- collisions_fokkerplanck.f90 -> Fokker-Planck


Hyper dissipation: 

- dissipation_hyper.f90 -> Can add hyper dissipation in k-space, zed, or vpa
